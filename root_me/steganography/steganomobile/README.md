After extraction of mobile data, the investigator got this sequence of numbers. Maybe a
phone number? [^1]

Solution
-----------------------------------------------------------------------------------------
At first, we are unfamiliar with this sequence of numbers, and search for it online.
Here, we can find out, that it is likely a type of SMS phone tap code cipher. And then,
it reminds us of how SMS texting used to be on the first mobile phones with a keyboard of
ten numeric keys. We had to press the same key multiple times to select the desired
letter. For this purpose, we can use an online tool like the dCode [^2] page, to then
enter the multi-tap mobile phone cipher text and decrypt it automatically. This results
in the word *CELLPHONE*, but for the validation password we have to enter it in lower
case.
