Find the password for the user *natas1* to log in at
`http://natas1.natas.labs.overthewire.org` for the next level. [^1]

Solution
-----------------------------------------------------------------------------------------
Since we are informed, that the password for the next level can be found on this site,
we open the page source of the *natas0* website with
`view-source:natas0.natas.labs.overthewire.org` and discover the *natas1* password in
the commented line underneath the body text.

[^1]: https://overthewire.org/wargames/natas/natas1.html
